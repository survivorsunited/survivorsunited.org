# Supported Mods

This setup supports a curated collection of Fabric-compatible mods. These mods enhance gameplay, improve performance, support anti-cheat enforcement, and add helpful features for exploration, inventory, multiplayer, and server stability.

## Mandatory Mods

This list is automatically synchronized from the latest modpack release. All mods listed here are required to connect to the Survivors United server.

## Core & Utility

- **CraterLib** – 2.1.5 – A library mod used by First Dark Development and HypherionSA Mods
- **Fabric API** – 0.136.1+1.21.8 – Core API module providing key hooks and intercompatibility features.
- **Fabric Language Kotlin** – 1.13.7+kotlin.2.2.21 – Fabric language module for Kotlin.
- **Forge Config API Port** – 21.8.2 – Provides NeoForge's & Forge's config systems to other modding ecosystems. Designed for a multi-loader architecture.
- **JamLib** – 1.3.5+1.21.8 – The platform-agnostic library used in all of JamCore's mods
- **libIPN** – 6.6.1 – Inventory Profiles Next Settings and GUI Library
- **Mod Menu** – 15.0.0 – Adds a mod menu to view the list of mods you have installed.
- **Placeholder API** – 2.7.2+1.21.8 – Simple api for mods allowing for user friendly formatting and universal placeholders!
- **ukulib** – 1.8.2+1.21.6 – small utility library used in my mods
- **YetAnotherConfigLib** – 3.7.1+1.21.6-fabric – YetAnotherConfigLib (yacl) is just that. A builder-based configuration library for Minecraft.

- **Custom Portals** – 3.9.1 – Build portals to anywhere out of any material!
- **Jade** – 19.3.2+fabric – Minecraft mod shows what you are looking at. (Hwyla fork)
- **Journeymap** – 1.21.8-6.0.0-beta.52 – JourneyMap: Real-time map in-game or in a web browser as you explore. JourneyMap API: v2.0.0. Built: 2025-09-05-15:23:03.
- **LambDynamicLights** – 4.8.6+1.21.8 – The most feature-complete dynamic lighting mod.

- **oωo** – 0.12.23+1.21.8 – yes its bad i know thanks

- **Waystones** – 21.8.5 – Teleport back to activated waystones. For Survival, Adventure or Servers.

## Multiplayer & Server

- **Open Parties and Claims** – 0.25.8 – This mod adds the ability to claim and to forceload world chunks, as well as create and manage player parties. It also gives server owners powerful controls over their players' usage of the mod's features. The mod's API allows other mods or plugins to easily interact with the parties and the chunk claims, both on the server and the client side. Xaero's Minimap mod and World Map mods are recommended in order to fully experience everything that this mod has to offer.
- **Pets Don't Die** – 1.1.5 – This data pack makes pets (tameable mobs) unable to die*.
- **Simple Voice Chat** – 1.21.8-2.6.6 – A working voice chat in Minecraft!

## Recycling

- **Diamond Recycling** – 1.3.2 – Recycle diamond tools, diamond armor, and smithing templates into diamonds through crafting.
- **MasterCutter** – 1.5.2-mc1.21.6 – The stonecutter is now more useful than ever with over 500 new recipes including woodcutting, enhanced base stonecutting, recycling and "un-crafting".
- **Netherite Recycling** – 1.4 – Recycle netherite tools and armor into netherite scraps via smelting or blasting.
- **Recycle Blast** – 1.61.0 – Recycle armors, tools, weapons, and other metal objects with the blast furnace
- **Sandstone Recycling** – 1.3 – Recycle sandstone into sand and other recipes
- **Stonecutter Salvage** – 1.7.4 – Salvage items in the Stonecutter
- **Wool to String** – 1.0.0 – This mod allows you to turn wool and carpet into string!

## Inventory Enhancements

- **Inventory HUD +** – 3.4.27 – This mod will show you your inventory while playing. Enjoy!
- **Inventory Management** – 1.6.0 – Sort and transfer items with the click of a button.  Built with RoundaLib Gradle plugin v1.0.0 and RoundaLib library v3.2.0.
- **ItemLocks** – 1.21.8-1.3.11 – Enables the client to lock slots to prevent accidental loss of items. Does not bypass server behavior.
- **Shulker Box Tooltip** – 5.2.12+1.21.8 – What's in my shulker box?
- **Stack Refill** – 4.9 – A quality of life mod which automatically refills the item in the hand of the player when using the final item if possible.

## Performance

- **FerriteCore** – 8.0.0 – Reduces memory usage
- **Lithium** – 0.18.1+mc1.21.8 – Lithium is a free and open-source optimization mod for Minecraft which makes a wide range of performance improvements to the game.
- **Sodium** – 0.7.3+mc1.21.8 – Sodium is a powerful rendering engine for Minecraft which greatly improves frame rates and micro-stutter, while fixing many graphical issues
- **spark** – 1.10.142 – spark is a performance profiling plugin/mod for Minecraft clients, servers and proxies.
