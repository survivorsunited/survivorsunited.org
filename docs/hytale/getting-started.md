# Hytale Getting Started

Welcome to Survivors United Hytale! This guide covers Early Access timing and how to connect.

## Early Access Details

Hytale Early Access launches **January 13, 2026**.  
Announcement: https://hytale.com/news/2025/11/hytale-early-access-january-13-2026

## Requirements

- **Hytale Early Access** (official release)
- **Register and Purchase**: Create an account and purchase a version before downloading: https://hytale.com/download
- **Internet connection** for multiplayer
- **Discord** for announcements and support: [${DISCORD_LOBBY}](${DISCORD_LOBBY})

## Server Details

- **Server Address**: `hytale.survivorsunited.org`
- **Server Name**: `Survivors United Hytale` (optional)

## How to Connect

1. Launch Hytale.
2. Open **Multiplayer**.
3. Choose **Add Server**.
4. Enter the server details above and save.
5. Join the server from your list.

## Mods

Hytale uses **server-side modding**, so players typically do **not** need to install mods on their client to join our server. The server delivers the experience automatically for consistency and security.

See the [Hytale Mods](./mods) page for the list of mods we are adding.

## Need Help?

Join Discord and check the dedicated Hytale channel for pinned updates, mod announcements, and support.
